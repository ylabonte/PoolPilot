# PoolPilot

One native iOS & Android app for the **ProCon.IP** and **Violet** pool controllers —
it auto-detects which controller you have, per connection profile.

### 🌐 Looking for the website?
**[poolpilot.eu](https://poolpilot.eu)** — features, help & FAQ, privacy and legal info.

### 🐞 Found a bug, or have an idea?
**[Open an issue](https://github.com/ylabonte/PoolPilot/issues/new/choose)** — bug reports,
feature requests and questions are welcome (German or English).

> ⚠️ Issues here are **public** — please never post passwords, controller credentials or
> IP addresses.
